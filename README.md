## üîß Mistral API (Optional) / Mistral API (–û–ø—Ü–∏–æ–Ω–∞–ª—å–Ω–æ)

Mistral API can **refine** the rule-based formula. It is **not required**.

Mistral API –º–æ–∂–µ—Ç **—É—Ç–æ—á–Ω–∏—Ç—å** —Ñ–æ—Ä–º—É–ª—É, —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞–Ω–Ω—É—é –ø–æ –ø—Ä–∞–≤–∏–ª–∞–º. **–ù–µ –æ–±—è–∑–∞—Ç–µ–ª–µ–Ω**.

### Setup / –ù–∞—Å—Ç—Ä–æ–π–∫–∞:
1. Get an API key at [Mistral AI](https://mistral.ai/)
2. Create `.env` in the project root:
   ```env
   MISTRAL_API_KEY=your_key_here
   ```
3. Enable in `config/uago_config.json`:
   ```json
   {
     "use_mistral_api": true,
     "mistral_model": "mistral-large-latest"
   }
   ```

> ‚ö†Ô∏è If the API fails or is disabled, UAGO-3C **automatically falls back** to its local rule-based system.  
> ‚ö†Ô∏è –ü—Ä–∏ –æ—à–∏–±–∫–µ –∏–ª–∏ –æ—Ç–∫–ª—é—á–µ–Ω–∏–∏ API —Å–∏—Å—Ç–µ–º–∞ **–∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–µ—Ä–µ–∫–ª—é—á–∞–µ—Ç—Å—è** –Ω–∞ –ª–æ–∫–∞–ª—å–Ω—É—é –≥–µ–Ω–µ—Ä–∞—Ü–∏—é.
